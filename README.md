# Sistema de Gestión de Restaurante

Proyecto final universitario - Sistema completo de gestión para restaurantes con Django, PostgreSQL y TailwindCSS.

## Características

- Autenticación con roles (Cliente, Mesero, Administrador)
- CRUD completo de platos, ingredientes y menú
- Sistema de pedidos y carrito de compras
- Reservas de mesas
- Control de inventario
- Dashboard con reportes y gráficos
- Exportación PDF y Excel
- Modo oscuro/claro
- Responsive design
- Confirmaciones por email

## Tecnologías

- Django 5.0
- PostgreSQL
- TailwindCSS 3.4
- Chart.js 4.4
- ReportLab
- OpenPyXL
- Alpine.js
